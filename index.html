Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates new branch and switches to it
  • git checkout main: takes you back to the main from the branch you are currently on
  • git add -A stages the changes to be made
  • git commit -m "name of commit" commits our changes
  • git pull origin main makes sure that we are up to date first meaning that local and remote branches are in sync
  • git push origin "name of branch" pushes up all of our changes to git

JavaScript

  • const = locking down a value for a variable so it cant be reassigned

  • each variable in an array are assigned a number value starting at 0 in order they are listed
